# LyricsAPI
Open the Solution file in Visual Studio and run the default web project.
You will be presented with a basic home page, click Artist Lookup in the header menu
Enter an artist name such as Coldplay or Nirvana, click Search
You will be presented with a dropdown list of potential matches to your artist search name, confirm your choice
You will then be presented with a list of the recordings for the chosen artist
